# SVG UI Mockups (V2 Only)

This folder contains the refined V2 mockups for the LearningLanguage app.

## Files
- `home_sessions.svg`
  - Home/session dashboard
  - language workspace switcher
  - progress-focused session cards
- `import_transcribe.svg`
  - session creation with two modes: `Import Audio` and `Create from Text`
  - Deepgram transcription (`/v1/listen`) and speech generation (`/v1/speak`) paths
  - explicit pipeline status for each mode
- `practice_sentence.svg`
  - sentence practice screen
  - draggable sentence progress bar
  - show/hide original transcript toggle
  - recording, user transcription, and word-level compare
- `settings_api_key.svg`
  - settings for Deepgram API key
  - save/validate/clear key actions
  - workspace isolation and behavior preferences
- `app_flow.svg`
  - high-level navigation and screen flow
  - includes both import-based and text-to-audio session entry

## Notes
- All V1 SVG files were removed.
- Design language is unified across pages (typography, spacing, controls, colors).
- Target format is iPhone portrait for page screens.
